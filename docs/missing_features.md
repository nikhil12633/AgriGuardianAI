\# Missing Features for AgriGuardianAI Capstone



\## Overview



AgriGuardianAI is a Google ADK-based multi-agent agricultural advisory system. The current implementation successfully supports weather analysis, soil analysis, market analysis, and agricultural recommendations.



The following features are recommended to achieve a top-level capstone submission.



\---



\# 1. High Priority Features



These features significantly improve the quality and completeness of the project.



\## 1.1 Persistent Farm Profiles



Current Status:



\* Farm information is not permanently stored.



Recommended:



\* Store farmer profiles including:



&#x20; \* Farmer name

&#x20; \* Location

&#x20; \* Farm size

&#x20; \* Soil type

&#x20; \* Preferred crops



Benefit:



\* Personalized recommendations.



\---



\## 1.2 Weather Forecast Support



Current Status:



\* Current weather retrieval only.



Recommended:



\* Add:



&#x20; \* 3-day forecast

&#x20; \* 7-day forecast

&#x20; \* rainfall prediction

&#x20; \* irrigation alerts



Benefit:



\* Better farming decisions.
