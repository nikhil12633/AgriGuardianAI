# AgriGuardian AI Architecture

## Coordinator Agent
Routes requests to specialized agents.

## Land Agent
Collects:
- State
- District
- Village
- Land Size
- Irrigation
- Current Crop

## Weather Agent
Provides:
- Temperature
- Rainfall
- Forecast
- Alerts

## Soil Agent
Provides:
- pH
- NPK
- Soil health

## Market Agent
Provides:
- Current prices
- Best market
- Revenue estimates

## Advisory Agent
Combines outputs from all agents and produces recommendations.
